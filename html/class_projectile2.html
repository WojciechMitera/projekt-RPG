Detailed Description

Represents a fast-moving projectile variant.

This class defines a lighter, faster projectile that deals less damage but travels at higher speed. It handles movement, collision detection, and automatic destruction.

void Projectile2.BodyCollision2 ( Node body)
inlineprivate

Handles collision with another body.

Applies damage to enemies and destroys the projectile on impact. Also destroys the projectile when it collides with environment surfaces such as walls, floor, or ceiling.

If no immediate collision condition is met, the projectile is scheduled for delayed destruction.

Parameters
bodyThe node that the projectile collided with.

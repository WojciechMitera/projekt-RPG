Detailed Description

Represents a projectile fired by enemies.

This class handles movement, collision detection, damage application to the player, and automatic destruction.

void Projectileenemy.BodyCollision2 ( Node body)
inlineprivate

Handles collision with another body.

Applies damage to the player when hit. Also destroys the projectile when it collides with environment surfaces such as walls, floor, or ceiling.

If no immediate collision condition is met, the projectile is scheduled for delayed destruction.

Parameters
bodyThe node that the projectile collided with.
